# Text-Editor-App-in-Python

A simple and user-friendly text editor built with **Python** and **Tkinter**. This application allows users to create, open, edit, and save text files efficiently with a minimalistic interface.

## Features

- **Create New File** – Start a new blank document.
- **Open File** – Open existing `.txt` files from your system.
- **Save File** – Save your work as a `.txt` file.
- **Exit Application** – Close the text editor safely.
- **Word Wrap Support** – Text wraps within the window for easy reading.
- **Customizable Font and Color** – Uses Ubuntu Sans font with red text color for readability.
